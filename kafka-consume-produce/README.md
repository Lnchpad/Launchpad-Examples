# Interacting with the Launchpad Platform via Kafka

## Getting Started

1. Create Virtual Env
	```bash
	$ python -m virtualenv <name you want>
	```

2. Activating the virtualenv

	```bash
	$ cd <name you want>
	$ source bin/activate
	```

3. Install Python Dependencies

	```bash
	$ pip -r requirements.txt
	```
